In the folder gym-FCTP/gym_FCTP/envs, the implementation of the FCTP environment can be found. The actions are however implemented in Java and called in Python with the help of Pyjnius. 

In the folder data_analysis, there are some basic functions for analyzing the performance of the rl agents, mainly by reading text files with results. There is also a file I made for creating latex tables from numpy arrays and one for plotting.
